# Fatigue-Guard
 
A wearable EMG sensor that detects muscle fatigue in real-time during strength training and alerts users before injury risk escalates.
 
## What It Does
 
Captures electrical signals from working muscles and triggers haptic and visual alerts when fatigue approaches dangerous levels. Uses dry electrodes, requires no special skin preparation, and costs a fraction of clinical EMG systems.
 
## Key Features
 
- Real-time muscle fatigue detection via surface EMG
- Personalised baseline for each training set
- Vibration motor and OLED display alerts
- Four to six-hour battery life
- Affordable component design
## Hardware
 
- ESP32 microcontroller
- INA128 amplifier
- Bandpass filter
- Dry electrode pads
- OLED display
- Vibration motor
- LiPo battery.
 
## How to Use
 
1. Apply dry electrode pads to the muscle
2. Wear the device on the wrist
3. Begin your training set
4. Device captures baseline automatically
5. REST NOW alert appears when fatigue is detected
## Status
 
Successfully tested in gym conditions with reliable fatigue detection and minimal false positives.
 
## Applications
 
Strength training, athletic performance monitoring, rehabilitation clinics, and injury recovery tracking.
